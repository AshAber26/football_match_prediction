# Football Match Prediction
## Project Description
The purpose of this project is to predict football match results using machine learning.
## Features used
- Performance difference
- Recent form (last 3 matches)
## Models used
- Logistic Regression
- Random Forest
- Support Vector Machines (SVM)
- K-Nearest Neighbours (KNN)
- Gradient Boosting
## Dataset
- A dataset from Kaggle was used, which was all fixtures from 2010-2021. 
- The dataset was preprocessed to fit the needs of my project, including removing matches (rows) and unnecessary features (columns).
